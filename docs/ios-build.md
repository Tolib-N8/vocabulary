# Сборка iOS-приложения и установка через AltStore

Собрать iOS-приложение на Linux локально нельзя — нужен Xcode. Поэтому сборка
происходит в облаке на GitHub Actions (macOS-раннер, бесплатно для публичных
репозиториев), а подписывает приложение сам AltStore вашим Apple ID.

## Шаг 1. Собрать IPA в GitHub Actions

1. Запушьте репозиторий на GitHub.
2. Откройте вкладку **Actions** → workflow **iOS unsigned IPA** → **Run workflow**
   (или создайте тег `v1.0.0` — сборка запустится сама).
3. Когда сборка закончится (~15 минут), скачайте артефакт
   `vocabulary-unsigned-ipa` — внутри будет `vocabulary-unsigned.ipa`.

IPA намеренно не подписан: AltStore при установке переподписывает приложение
вашим бесплатным Apple ID, поэтому сертификаты разработчика не нужны.

## Шаг 2. Установить AltStore на iPhone

1. На компьютере установите **AltServer** (https://altstore.io) — есть версии
   для Windows и macOS. На Linux используйте открытую альтернативу
   **AltServer-Linux** или **SideStore** (https://sidestore.io) — SideStore
   после первой установки вообще не требует компьютера.
2. Подключите iPhone кабелем и установите AltStore через AltServer
   (Install AltStore → введите Apple ID).
3. На iPhone: Настройки → Основные → VPN и управление устройством →
   доверьте сертификату вашего Apple ID.

## Шаг 3. Установить приложение

1. Передайте `vocabulary-unsigned.ipa` на iPhone (AirDrop, файлы, облако).
2. Откройте AltStore → **My Apps** → **+** → выберите ipa-файл.
3. AltStore подпишет и установит приложение.

## Ограничения бесплатного Apple ID

- Приложение работает **7 дней**, потом его нужно обновить (Refresh) в
  AltStore — при запущенном AltServer в той же Wi-Fi-сети это одна кнопка,
  SideStore умеет обновлять без компьютера.
- Одновременно можно держать не больше **3 приложений**, установленных таким
  способом.
- Первый запуск: разрешите доступ к микрофону и распознаванию речи — без них
  режим shadowing будет работать в режиме самооценки.
