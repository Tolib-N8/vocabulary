# Словарь — изучение английских слов

Приложение для заучивания английских слов: карточки с интервальными
повторениями (FSRS), тренировка правописания и shadowing с распознаванием
речи. Работает в браузере (десктоп) и на iPhone (через AltStore).

## Режимы

- **🃏 Карточки** — слово ↔ перевод, оценки «Снова / Трудно / Хорошо / Легко»,
  расписание повторений считает [ts-fsrs](https://github.com/open-spaced-repetition/ts-fsrs).
- **✍️ Правописание** — приложение произносит слово, вы печатаете его;
  ошибки подсвечиваются посимвольно.
- **🎙 Shadowing** — приложение озвучивает слово и пример, вы повторяете в
  микрофон; распознавание речи сравнивает сказанное с эталоном и ставит
  процент совпадения. Если распознавание недоступно — самооценка.

Прогресс по каждому режиму считается отдельно. Встроенные наборы слов
A1–B2 (~500 слов с переводом, транскрипцией и примером) плюс свои слова.
Все данные хранятся локально (SQLite), есть экспорт/импорт в JSON.

## Запуск (веб)

```bash
npm install
npm run web
```

Открывайте в **Chrome** — там работают и озвучка, и распознавание речи
(Web Speech API). В Firefox распознавание недоступно, shadowing перейдёт в
режим самооценки.

## Тесты

```bash
npm test
```

## iOS (AltStore)

Сборка ipa происходит в GitHub Actions, подписывает приложение сам AltStore.
Подробная инструкция: [docs/ios-build.md](docs/ios-build.md).

## Структура

```
src/app/        экраны (expo-router)
src/db/         SQLite: схема, репозитории, сиды, бэкап
src/srs/        FSRS-движок и очередь сессии
src/speech/     TTS, распознавание речи (web/native), scoring
assets/wordsets встроенные наборы слов A1–B2
```
